## Countries API

This is a project that aims to give information about countries.

### Built With:

- Next.js,
- TypeScript,
- Tailwind CSS,
- daisyUI,
- Next.js API Routes,
- Axios.

### Used Methods/Hooks:

- useState,
- useEffect,
- map(),
- Conditional Rendering (&& and ?),
- Array destructuring,
- POST API route,
- POST method.
